# API Endpoints

---

## Users

### GET /users

Returns all users.

### POST /users

Creates a new user.

---

## Food Items

### GET /food-items

Returns all food items.

### POST /food-items

Creates a new food item.

---

## Purchases

### GET /purchases

Returns all purchase records.

### POST /purchases

Creates a purchase.

---

## Analytics

### GET /analytics/monthly-spending

Returns total spending grouped by user.
